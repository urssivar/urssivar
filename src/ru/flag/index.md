# Кайтагский флаг

![Swallowtail](https://raw.githubusercontent.com/urssivar/flag/refs/heads/main/swallowtail.svg)

[Флаг](https://github.com/urssivar/flag) **кайтагского** народа (**Хайдаҡӏла**) — синее полотнище с белой печатью в центре.

Стилизованная **куфическая монограмма** из пантеона правителей Кайтага символизирует многовековую государственность.

Чуть выше и правее — белый четырёхлепестковый **цветок марены красильной**, некогда в обилии выращивавшейся в Кайтаге.

**Глубокий синий** — традиционный фон кайтагской вышивки.

Пропорции 2:3 (ласточкин хвост 1:2). Цвет #0047AB / PMS 300 C.
